# Simple Pong

A minimal Pong game built with HTML, CSS and JavaScript.

Features:
- Left paddle controlled by mouse and Up/Down arrow keys
- Computer AI controls the right paddle
- Ball bounces off paddles and walls
- Scoreboard and win condition (first to 10)
- Pause and restart buttons

How to run:
1. Open index.html in a modern browser.
2. Move your mouse over the canvas or use Up/Down keys to move the left paddle.

Files:
- index.html — markup and canvas
- styles.css — styling
- script.js — game logic and rendering

Enjoy!
